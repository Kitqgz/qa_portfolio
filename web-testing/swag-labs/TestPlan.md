1. Общая информация

Название проекта:
Swag Labs Login Testing

---------------------------------

2. Цель тестирования:
Проверить корректность работы авторизации пользователей и обработку системой корректных и некорректных учетных данных.

---------------------------------

3. Область тестирования

Будет протестировано:
✔ форма авторизации
✔ поля Username
✔ поле Password
✔ кнопка Login
✔ сообщения об ошибках

Не тестируется:
✘ каталог товаров
✘ корзина
✘ оформление заказа

---------------------------------

4. Виды тестирования
✔ Functional Testing
✔ Smoke Testing
✔ Positive Testing
✔ Negative Testing

Подход:
✔ Black-box Testing

---------------------------------

7. Критерии входа

Приложение доступно по указанному URL.
Тестовое окружение подготовлено.
Доступны тестовые учетные записи.
Google Chrome установлен и работает корректно.

---------------------------------

8. Критерии выхода

Выполнены все тест-кейсы.
Все критические дефекты закрыты или согласованы.
Подготовлен Test Summary Report.

---------------------------------

9. Риски

Недоступность тестового стенда
Изменение функциональности сайта
Проблемы с интернет-соединением
