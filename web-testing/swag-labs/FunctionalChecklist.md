# Swag Labs — Functional Checklist

## Scope

Функциональное, позитивное, негативное и базовое UI-тестирование формы авторизации Swag Labs.

## Test environment

- URL: https://www.saucedemo.com/
- Browser: Google Chrome
- Test type: Manual functional testing
- Approach: Black-box testing

## Status legend

- ⬜ Not Run
- ✅ Passed
- ❌ Failed
- ⚠️ Blocked
